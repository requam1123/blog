## Step 3 选择一个静态网站生成器    
静态网站生成器（Static Site Generator, SSG）是一种工具，可以将纯文本文件（通常是Markdown格式）转换为静态HTML文件，从而生成一个完整的网站。相比于传统的动态网站，静态网站更快、更安全，因为它们不依赖于服务器端的数据库或脚本。

**这里推荐mkdocs**
